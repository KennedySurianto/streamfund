# Streamfund 🚀

A real-time creator donation platform featuring instant OBS stream alerts.

![Streamfund Donation Simulation](https://i.makeagif.com/media/4-02-2026/AXv2Si.gif)

## ✨ Features

* **Live OBS Overlays:** Instant, real-time stream alerts powered by Server-Sent Events (SSE).
* **Native Text-to-Speech (TTS):** Automated Indonesian voice alerts that read donation amounts and messages directly into the stream audio mixer.
* **Creator Profiles:** Clean, public-facing checkout pages where supporters can leave custom messages.

## 🛠️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Go, Gin Framework, GORM
* **Database:** PostgreSQL via Docker
